# human-history
# 人类文明演进史 - 完整沉浸版

> 从第一缕意识之火到数字时代的黎明，穿越700万年的时光长河。

## 项目简介

这是一个单页面的交互式历史时间线网站，旨在通过视觉化和互动的方式展示人类文明从起源到现代的完整历程。项目包含 **500+ 个关键历史事件**，覆盖史前时代、古代文明、古典时代、中世纪、文艺复兴、工业革命、现代及当代等八大历史时期。

本项目无需后端支持，仅需一个 HTML 文件即可运行，集成了动态背景、响应式布局和丰富的交互效果。

## 功能特性

### 1. 内容丰富
- **八大时代**：从史前时代到当代，每个时期都有独特的视觉主题。
- **500+ 事件**：涵盖科技、科学、文化、战争、人物等五大类别。
- **详细解读**：点击事件卡片可查看详细描述和历史意义。

### 2. 视觉沉浸
- **时代主题背景**：
  - 史前时代：岩石纹理与原始图案
  - 古代文明：波浪纹与小麦元素
  - 现代与当代：科幻网格与扫描线特效
- **动态粒子系统**：基于 Canvas 的动态连线粒子背景，增加科技感与神秘感。
- **线性时间轴**：贯穿全页的渐变色时间轴，直观展示历史进程。

### 3. 交互体验
- **滚动动画**：事件卡片随滚动渐入显示。
- **详情模态框**：点击卡片弹出详情，支持键盘 ESC 关闭。
- **响应式设计**：完美适配桌面端与移动端。
- **快速导航**：桌面端侧边栏快速跳转，移动端汉堡菜单导航。

## 技术栈

- **核心**：HTML5, CSS3, JavaScript (ES6+)
- **样式**：Tailwind CSS (CDN)
- **图标**：Font Awesome 6.4.0
- **字体**：Google Fonts (Cinzel, Noto Sans SC)
- **图形**：Canvas API (粒子背景)

## 自定义指南

### 修改历史数据
在 `<script>` 标签内找到 `events` 数组，按照以下格式添加或修改事件：

javascript
const events = [
{
year: ‘2023年’, // 时间点
title: ‘事件名称’, // 标题
desc: ‘简短描述…’, // 描述
era: ‘contemporary’, // 所属时代
type: ‘tech’, // 类别
badge: ‘第一次’, // 徽章文字
detail: ‘详细说明…’ // 详情（可选）
},
// 更多事件…
];


**时代 对应值**：
- 史前时代: `prehistoric`
- 古代文明: `ancient`
- 古典时代: `classical`
- 中世纪: `medieval`
- 文艺复兴: `renaissance`
- 工业革命: `industrial`
- 现代: `modern`
- 当代: `contemporary`

**类别 对应值**：
- 技术发明: `tech`
- 科学发现: `science`
- 历史人物: `person`
- 文化事件: `culture`
- 战争冲突: `war`

### 调整视觉风格
- **时代配色**：修改 `tailwind.config` 中的颜色变量。
- **背景图案**：修改 CSS 中 `.era-[时代名]` 的 `background-image` 属性。
- **粒子数量**：修改 `initParticles()` 函数中的循环次数。
